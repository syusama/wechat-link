## English

### Repository Description

`An unofficial Python SDK for iLink-compatible Weixin Bot integration, focused on protocol handling, media delivery, and a lightweight relay.`

### Alternate Description

`SDK-first Python toolkit for iLink-compatible Weixin Bot APIs, including polling, messaging, media upload, and a lightweight relay.`

`Turn the core iLink / Weixin Bot protocol into a clean, reusable, embeddable Python SDK.`

`wechat-link` is an unofficial open-source project. Instead of growing into an all-in-one bot platform, it focuses on the reusable foundation: protocol primitives, media delivery flows, and a small relay layer that can be integrated into your own apps, LLM systems, or workflow backends.
